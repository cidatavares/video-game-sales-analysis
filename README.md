# 🎮 Video Game Sales Analysis

## 📌 Sobre o projeto

Projeto desenvolvido durante o Bootcamp de Data Analytics da TripleTen.

O objetivo foi analisar dados históricos da indústria de jogos eletrônicos para identificar padrões capazes de prever o sucesso comercial de novos jogos.

---

## 📊 Objetivos

- Limpeza e preparação dos dados
- Análise exploratória (EDA)
- Comparação entre plataformas
- Comparação entre gêneros
- Análise por regiões
- Testes de hipóteses estatísticas

---

## 🛠 Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

---

## 📈 Principais análises

✔ Evolução das vendas por plataforma

✔ Distribuição das vendas globais

✔ Relação entre avaliações e vendas

✔ Comparação entre gêneros

✔ Perfil dos consumidores por região

✔ Influência da classificação ESRB

✔ Testes estatísticos
